## ğŸ› TROUBLESHOOTING

### Website won't open?
- Make sure `index.html` is in the correct folder
- Use a modern browser
- Check file path

### Styles not showing?
- Refresh the page (Ctrl+F5)
- Check `styles.css` is in same folder
- Clear browser cache

### Data not saving?
- Enable local storage in browser
- Check browser isn't in private mode
- Try a different browser

### Menu items not showing?
- Check `script.js` is in same folder
- Open browser console (F12) to check for errors
- Refresh page
